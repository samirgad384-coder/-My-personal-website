## 📸 Overview

A sleek, fully responsive personal portfolio built with pure HTML, CSS & JavaScript — no frameworks, no dependencies. Designed with a premium aesthetic inspired by the colors of **Tsinghua University**, reflecting my ambition to study there through the **CSC Scholarship**.

---

## ✨ Features

| Feature | Details |
|---------|---------|
| 🎨 **Premium Design** | Tsinghua red & gold palette with a subtle dot-grid background |
| 💎 **Glassmorphism Header** | Frosted-glass navbar with backdrop blur |
| ✨ **Smooth Animations** | Fade-in-up entrance animations on page load |
| 📱 **Fully Responsive** | Optimized for mobile, tablet & desktop |
| ⚡ **Zero Dependencies** | Pure HTML/CSS/JS — blazing fast & lightweight |
| 🔠 **Arabic Typography** | Amiri (headings) + Cairo (body) from Google Fonts |
| 🖱️ **Custom Scrollbar** | Gold-themed scrollbar for a polished touch |

## 🌍 Deploy on GitHub Pages

1. Go to your repository **Settings**
2. Click **Pages** in the sidebar
3. Under **Source**, select `Deploy from a branch`
4. Choose **Branch: main** and **Folder: / (root)**
5. Hit **Save** ✅

Your site will be live in ~2 minutes at:
`https://samirgad384-coder.github.io/-My-personal-website`
